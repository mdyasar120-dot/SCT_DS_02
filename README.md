# Titanic Data Analysis 🚢

## 📌 Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to understand factors affecting passenger survival.

## 🛠️ Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Analysis Performed
- Data Cleaning (handling missing values, duplicates)
- Survival analysis by gender and class
- Age and fare distribution
- Correlation analysis
- Pairplot visualization

## 🔍 Key Insights
- Females had significantly higher survival rates than males
- 1st class passengers had better survival chances
- Higher fare is associated with higher survival probability
- Younger passengers had slightly better survival rates
